# HTML CSS Tester

이 프로젝트는 HTML, CSS를 테스트하기 위한 프로젝트입니다.
각종 프로젝트에서 사용하는 web page를 작성하고 수정하여 최선을 다한 고품질의 웹페이지를 만들기 위함이다.
각종 css와 javascript를 작성하고 테스트하는 것을 목적으로 한다.


## 기술 스택

- HTML5, CSS3
- 기본적으로 CSS framework로는 [tailwindcss](https://tailwindcss.com/)를 사용한다.
- javascript는 [alpinejs](https://alpinejs.dev/)을 사용한다.

**주의** 
    - 각 해당 프로젝트의 성격에 따라서 기술스택은 바뀔 수 있다. 
    - 주로 CDN을 사용한다.

## 폴더 및 파일명명 규칙

- 폴더명은 영문 소문자와 대쉬(-)로 작성한다.
- 파일명은 영문 소문자와 대쉬(-)로 작성한다.

- apps 폴더는 각각의 프로젝트를 위한 폴더로 구성한다.
- apps 하위에 각 프로젝트의 폴더가 생성된다.
