# 텍스트 뷰어 (text.html)

## 개요

평문 텍스트 문서 뷰어입니다. 텍스트 파일을 렌더링하며 라인 번호, 검색, 폰트 조정, 배경색 선택, 줄간 조정, 다크모드 등의 기능을 제공합니다.

**주요 특징:**

- 구문 강조 없음 (순수 텍스트 표시)
- 라인 번호 자동 표시
- 실시간 검색 기능
- 사용자 선호도 저장 (폰트, 크기, 배경색, 다크모드 등)

---

## Header 버튼들 (상단 우측)

> 데스크톱 화면에서 파일명 우측에 표시됩니다.

| 버튼 | 설명 |
|------|------|
| **전체 복사** | 문서 전체 텍스트를 클립보드에 복사합니다. |
| **원본 다운로드** | 원본 텍스트 파일을 다운로드합니다. |
| **문서 정보 토글** | 파일 크기, 라인 수, 문자 수, 단어 수, 인코딩 정보를 표시/숨김합니다. |

---

## Control Bar 버튼들 (문서 위)

> 문서 상단에 위치한 제어 바에서 다양한 설정을 조절할 수 있습니다.

### 텍스트 표시 설정

| 버튼 | 설명 |
|------|------|
| **커서라인** | 현재 커서가 있는 라인을 하이라이트합니다. (체크박스) |
| **줄바꿈** | 긴 라인의 줄바꿈 여부를 토글합니다. (체크박스) |

### 검색 기능

| 버튼 | 설명 |
|------|------|
| **검색** | 텍스트 내에서 실시간 검색을 수행합니다. |

### 폰트 및 외관 설정

| 버튼 | 설명 |
|------|------|
| **폰트 작게 / 기본 / 크게** | 폰트 크기를 조정합니다. (기본값: 14px) |
| **폰트 선택** | 드롭다운으로 폰트를 선택합니다.<br/>- Noto Sans KR<br/>- 맑은 고딕<br/>- 서울 남산체<br/>- Courier New<br/>- Monaco |
| **배경색 선택** | 배경색을 선택합니다.<br/>- 흰색 (기본)<br/>- 라이트 그레이<br/>- 라이트 옐로우<br/>- 라이트 블루<br/>- 라이트 그린 |
| **줄간 작게 / 기본 / 크게** | 텍스트 줄 간격을 조정합니다. (기본값: 1.6) |

### 추가 기능

| 버튼 | 설명 |
|------|------|
| **전체화면** | 전체 화면 모드로 전환합니다. (⛶) |
| **다크모드** | 어두운 배경 테마를 토글합니다. (🌙) |
| **설정 저장** | 현재 설정(폰트, 크기, 배경색, 줄간, 다크모드 등)을 localStorage에 저장합니다. |
| **설정 초기화** | 저장된 설정을 초기값으로 리셋합니다. |
| **도움말** | 사용 팁을 표시하는 모달을 엽니다. |

## Hotkey (키보드 단축키)

| 단축키 | 설명 |
|--------|------|
| **Ctrl+S** (Windows/Linux) | 원본 파일 다운로드 |
| **Cmd+S** (macOS) | 원본 파일 다운로드 |
| **Home** | 페이지 맨 위로 스크롤 |
| **F4** | 전체화면 모드 토글 |
| **Escape** | 도움말 모달 닫기 |
| **Enter** (검색창) | 검색 입력창에서 다음 검색 결과로 이동 |
| **Delete** (검색창) | 검색창 초기화 |

---

## 팁

- **폰트 크기 조정**: 기본값 14px에서 시작하며, 각 버튼으로 1px씩 증감합니다.
- **설정 자동 로드**: 페이지 로드 시 저장된 설정이 자동으로 적용됩니다.
- **검색**: Enter 키로 다음 검색 결과를 찾거나, Delete 키로 검색을 초기화할 수 있습니다.
- **전체 복사**: 마우스로 선택할 수도 있지만, "전체 복사" 버튼으로 한 번에 복사할 수 있습니다.
