# 소스코드 뷰어 (source.html)

## 개요

소스코드 뷰어입니다. 프로그래밍 언어별 문법 강조(Syntax Highlighting)를 지원하며, 라인 번호, 페이지 뷰 모드, 검색, 폰트 조정, 배경색 선택, 줄간 조정, 다크모드 등의 기능을 제공합니다.

**주요 특징:**

- Highlight.js를 통한 다양한 프로그래밍 언어 문법 강조
- 자동 언어 감지
- 다중 페이지 뷰 모드 (1열, 2열, 3열)
- 라인 번호 자동 표시
- 실시간 검색 기능
- 사용자 선호도 저장

---

## Header 버튼들 (상단 우측)

> 데스크톱 화면에서 파일명 우측에 표시됩니다.

| 버튼 | 설명 |
|------|------|
| **복사하기** | 소스코드 전체를 클립보드에 복사합니다. |
| **원본 다운로드** | 원본 소스코드 파일을 다운로드합니다. |
| **문서 정보 토글** | 파일 크기, 라인 수, 비어있지 않은 라인, 문자 수, 프로그래밍 언어 정보를 표시/숨김합니다. |

---

## Control Bar 버튼들 (문서 위)

> 문서 상단에 위치한 제어 바에서 다양한 설정을 조절할 수 있습니다.

### 텍스트 표시 설정

| 버튼 | 설명 |
|------|------|
| **커서라인** | 현재 커서가 있는 라인을 하이라이트합니다. (체크박스) |

### 페이지 뷰 모드

| 버튼 | 설명 |
|------|------|
| **페이지 뷰** | 1페이지, 2페이지(2열), 3페이지(3열) 레이아웃으로 전환합니다.<br/>- 1: 단일 열 표시<br/>- 2: 2개 열로 분할 표시<br/>- 3: 3개 열로 분할 표시 |

### 검색 기능

| 버튼 | 설명 |
|------|------|
| **검색** | 소스코드 내에서 실시간 검색을 수행합니다. |

### 폰트 및 외관 설정

| 버튼 | 설명 |
|------|------|
| **폰트 작게 / 기본 / 크게** | 폰트 크기를 조정합니다. (기본값: 14px) |
| **폰트 선택** | 드롭다운으로 코딩 폰트를 선택합니다.<br/>- D2Coding (기본)<br/>- 나눔고딕코딩<br/>- JetBrains Mono<br/>- Fira Code<br/>- Consolas |
| **배경색 선택** | 배경색을 선택합니다.<br/>- 라이트 그레이 (#f6f8fa)<br/>- 흰색<br/>- 크림색<br/>- 라이트 블루<br/>- 라이트 그린 |
| **줄간 작게 / 기본 / 크게** | 라인 높이를 조정합니다. (기본값: 1.6) |

### 추가 기능

| 버튼 | 설명 |
|------|------|
| **전체화면** | 전체 화면 모드로 전환합니다. (⛶) |
| **다크모드** | 어두운 배경 테마를 토글합니다. (🌙) |
| **설정 저장** | 현재 설정(폰트, 크기, 배경색, 줄간, 다크모드 등)을 localStorage에 저장합니다. |
| **설정 초기화** | 저장된 설정을 초기값으로 리셋합니다. |
| **도움말** | 사용 팁을 표시하는 모달을 엽니다. |

## Hotkey (키보드 단축키)

| 단축키 | 설명 |
|--------|------|
| **Ctrl+S** (Windows/Linux) | 원본 파일 다운로드 |
| **Cmd+S** (macOS) | 원본 파일 다운로드 |
| **Home** | 페이지 맨 위로 스크롤 |
| **F4** | 전체화면 모드 토글 |
| **Escape** | 도움말 모달 닫기 |
| **Enter** (검색창) | 검색 입력창에서 다음 검색 결과로 이동 |

## 팁

- **언어 감지**: 파일 확장자를 기반으로 프로그래밍 언어가 자동으로 감지되고 문법 강조가 적용됩니다.
- **페이지 뷰 모드**: 2열 또는 3열 모드는 긴 코드를 여러 컬럼으로 나누어 표시하므로 모니터 너비가 크면 유용합니다.
- **다크모드와 배경색**: 다크모드와 배경색 선택은 별도로 작동합니다. 다크모드를 활성화하면 배경색은 무시됩니다.
- **설정 자동 로드**: 페이지 로드 시 저장된 설정이 자동으로 적용됩니다.
- **복사하기**: 선택이 가능하지만, "복사하기" 버튼으로 한 번에 전체를 복사할 수 있습니다.
