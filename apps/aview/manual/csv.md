# CSV 뷰어 (csv.html)

## 개요

CSV/표 데이터 뷰어입니다. Tabulator.js를 사용하여 표 형식의 데이터를 렌더링합니다. 페이지네이션, 컬럼 선택, 정렬, 검색, 엑셀 내보내기, 폰트 조정, 다크모드 등의 기능을 제공합니다.

**주요 특징:**

- Tabulator.js를 통한 대용량 데이터 처리
- 자동 페이지네이션
- 동적 컬럼 표시/숨김
- 클릭으로 컬럼 정렬
- 실시간 검색 기능
- XLSX 형식 내보내기
- 사용자 선호도 저장

---

## Header 버튼들 (상단 우측)

> 데스크톱 화면에서 파일명 우측에 표시됩니다.

| 버튼 | 설명 |
|------|------|
| **원본 다운로드** | 원본 CSV 파일을 다운로드합니다. |
| **문서 정보 토글** | 파일 크기, 총 행 수, 총 열 수, 구분자 정보를 표시/숨김합니다. |

---

## Control Bar 버튼들 (테이블 위)

> 테이블 상단에 위치한 제어 바에서 다양한 설정을 조절할 수 있습니다.

### 폰트 설정

| 버튼 | 설명 |
|------|------|
| **폰트 선택** | 드롭다운으로 폰트를 선택합니다.<br/>- Noto Sans KR (기본)<br/>- 맑은 고딕<br/>- 서울 남산체<br/>- Courier New |
| **폰트 작게 / 기본(14px) / 크게** | 폰트 크기를 조정합니다. |

### 페이지 설정

| 버튼 | 설명 |
|------|------|
| **페이지 크기** | 드롭다운으로 한 페이지당 행 수를 선택합니다.<br/>- 10행 (기본)<br/>- 25행<br/>- 50행<br/>- 100행<br/>- 전체 (페이지네이션 비활성화) |

### 검색 기능

| 버튼 | 설명 |
|------|------|
| **검색** | 테이블 데이터를 실시간으로 검색합니다. |

### 컬럼 설정

| 버튼 | 설명 |
|------|------|
| **컬럼 선택** | 표시할 컬럼을 선택하는 패널을 엽니다. 불필요한 컬럼을 숨길 수 있습니다. |
| **컬럼 너비 자동 맞춤** | 각 컬럼의 너비를 자동으로 조정하여 콘텐츠에 맞춥니다. |

### 내보내기

| 버튼 | 설명 |
|------|------|
| **엑셀로 내보내기** | 현재 CSV 데이터를 XLSX 파일로 다운로드합니다. |

### 추가 기능

| 버튼 | 설명 |
|------|------|
| **전체화면** | 전체 화면 모드로 전환합니다. (⛶) |
| **다크모드** | 어두운 배경 테마를 토글합니다. (🌙) |
| **설정 저장** | 현재 설정(폰트, 페이지 크기, 다크모드 등)을 localStorage에 저장합니다. |
| **설정 초기화** | 저장된 설정을 초기값으로 리셋합니다. |
| **도움말** | 사용 팁을 표시하는 모달을 엽니다. |

## Hotkey (키보드 단축키)

| 단축키 | 설명 |
|--------|------|
| **Ctrl+S** (Windows/Linux) | 원본 CSV 파일 다운로드 |
| **Cmd+S** (macOS) | 원본 CSV 파일 다운로드 |
| **Home** | 페이지 맨 위로 스크롤 |
| **F4** | 전체화면 모드 토글 |
| **Escape** | 도움말 모달 닫기 |

## 팁

- **컬럼 정렬**: 테이블의 컬럼 헤더를 클릭하면 해당 컬럼으로 정렬됩니다. 다시 클릭하면 역순 정렬됩니다.
- **페이지네이션**: 기본값은 10행씩 표시됩니다. 데이터가 많으면 페이지 크기를 조정하여 로딩 속도를 개선할 수 있습니다.
- **대용량 파일**: Tabulator.js의 가상 스크롤 기능으로 대용량 CSV 파일도 빠르게 렌더링됩니다.
- **컬럼 숨김**: 컬럼 선택 기능으로 필요한 컬럼만 표시하여 테이블을 더 읽기 쉽게 만들 수 있습니다.
- **검색**: 검색은 현재 페이지뿐만 아니라 전체 데이터를 대상으로 수행됩니다.
- **엑셀 내보내기**: XLSX 형식으로 내보내지므로 Microsoft Excel, Google Sheets 등에서 바로 열 수 있습니다.
- **설정 자동 로드**: 페이지 로드 시 저장된 설정이 자동으로 적용됩니다.
