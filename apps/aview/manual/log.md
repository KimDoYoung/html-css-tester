# 로그 뷰어 (log.html)

## 개요

로그 파일 뷰어입니다. 로그 파일의 로그 레벨(DEBUG, INFO, WARN, ERROR, SYSTEM)별로 색상 강조 기능을 제공합니다. 검색, 폰트 조정, 배경색 선택, 다크모드, 로그 레벨 필터링, 전체 복사 등의 기능을 제공합니다.

**주요 특징:**

- 로그 레벨별 자동 색상 강조
- 로그 레벨 필터링
- 실시간 검색 기능
- 라인 번호 자동 표시
- 전체 복사 기능
- 사용자 선호도 저장

## Header 버튼들 (상단 우측)

> 데스크톱 화면에서 파일명 우측에 표시됩니다.

| 버튼 | 설명 |
|------|------|
| **원본 다운로드** | 원본 로그 파일을 다운로드합니다. |
| **문서 정보 토글** | 파일 크기, 라인 수, 문자 수, 단어 수, 인코딩 정보를 표시/숨김합니다. |

## Control Bar 버튼들 (문서 위)

> 문서 상단에 위치한 제어 바에서 다양한 설정을 조절할 수 있습니다.

### 검색 기능

| 버튼 | 설명 |
|------|------|
| **검색** | 로그 내에서 실시간 검색을 수행합니다. |

### 텍스트 표시 설정

| 버튼 | 설명 |
|------|------|
| **줄바꿈** | 긴 라인의 줄바꿈 여부를 토글합니다. (체크박스) |

### 폰트 및 외관 설정

| 버튼 | 설명 |
|------|------|
| **폰트 작게 / 기본(14px) / 크게** | 폰트 크기를 조정합니다. |
| **폰트 선택** | 드롭다운으로 폰트를 선택합니다.<br/>- Noto Sans KR<br/>- D2Coding (기본 모노스페이스)<br/>- 나눔고딕코딩<br/>- Cascadia Code |
| **배경색 선택** | 배경색을 선택합니다.<br/>- 흰색 (기본)<br/>- 라이트 그레이<br/>- 라이트 옐로우<br/>- 라이트 블루<br/>- 라이트 그린 |
| **줄간 작게 / 기본 / 크게** | 라인 높이를 조정합니다. (기본값: 1.6) |

### 로그 필터링

| 버튼 | 설명 |
|------|------|
| **로그 레벨 컬러링** | 로그 레벨별 색상 강조 여부를 선택합니다.<br/>- **DEBUG** 🔵 (파란색)<br/>- **INFO** 🟢 (초록색)<br/>- **WARN** 🟡 (노란색)<br/>- **ERROR** 🔴 (빨간색)<br/>- **SYSTEM** ⚙️ (회색) |

### 추가 기능

| 버튼 | 설명 |
|------|------|
| **전체 복사** | 로그 파일 전체를 클립보드에 복사합니다. |
| **전체화면** | 전체 화면 모드로 전환합니다. (⛶) |
| **다크모드** | 어두운 배경 테마를 토글합니다. (🌙) |
| **설정 저장** | 현재 설정(폰트, 크기, 배경색, 줄간, 다크모드, 로그 레벨 필터 등)을 localStorage에 저장합니다. |
| **설정 초기화** | 저장된 설정을 초기값으로 리셋합니다. |
| **도움말** | 사용 팁을 표시하는 모달을 엽니다. |

## Hotkey (키보드 단축키)

| 단축키 | 설명 |
|--------|------|
| **Ctrl+S** (Windows/Linux) | 원본 로그 파일 다운로드 |
| **Cmd+S** (macOS) | 원본 로그 파일 다운로드 |
| **Home** | 페이지 맨 위로 스크롤 |
| **F4** | 전체화면 모드 토글 |
| **Escape** | 도움말 모달 닫기 |
| **Enter** (검색창) | 검색 입력창에서 다음 검색 결과로 이동 |

## 팁

- **로그 레벨 색상**:
  - **DEBUG** (파란색): 개발자용 디버그 메시지
  - **INFO** (초록색): 일반 정보 메시지
  - **WARN** (노란색): 경고 메시지
  - **ERROR** (빨간색): 오류 메시지
  - **SYSTEM** (회색): 시스템 메시지

- **필터링**: 로그 레벨 선택 박스로 특정 레벨의 로그만 표시할 수 있습니다. 예를 들어, ERROR만 선택하면 에러 로그만 빨간색으로 강조됩니다.

- **검색**: 검색은 모든 로그 라인을 대상으로 수행됩니다. 예를 들어, "Exception"을 검색하면 해당 단어가 있는 모든 라인을 찾을 수 있습니다.

- **전체 복사**: 마우스로 선택할 수도 있지만, "전체 복사" 버튼으로 한 번에 복사할 수 있습니다.

- **줄바꿈**: 긴 로그 라인이 있으면 줄바꿈을 활성화하여 가독성을 높일 수 있습니다.

- **다크모드**: 장시간 로그 분석을 할 때 다크모드를 사용하여 눈 피로를 줄일 수 있습니다.

- **설정 자동 로드**: 페이지 로드 시 저장된 설정이 자동으로 적용됩니다.
