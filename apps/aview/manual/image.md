# 이미지 뷰어 (image.html)

## 개요

이미지 뷰어입니다. Viewer.js 라이브러리를 사용하여 이미지를 렌더링합니다. 줌(확대/축소), 회전, 좌우/상하 반전, 원본 크기 보기, 초기화, 최대화, 새 창 열기, 원본 다운로드 등의 기능을 제공합니다.

**주요 특징:**

- Viewer.js를 통한 강력한 이미지 조작
- 부드러운 줌 및 팬(이동) 기능
- 90도 회전 지원
- 좌우/상하 반전 기능
- 원본 크기 보기 (1:1)
- 전체 화면 모드
- 이미지 정보 표시

---

## Header 버튼들 (상단 우측)

> 데스크톱 화면에서 파일명 우측에 표시됩니다.

| 버튼 | 설명 |
|------|------|
| **새 창에서 열기** | 이미지를 새로운 브라우저 탭에서 엽니다. |
| **원본 다운로드** | 원본 이미지 파일을 다운로드합니다. |
| **문서 정보 토글** | 파일 크기, 이미지 크기(해상도), 포맷, 색상 모델, 비트 깊이, 투명도 정보를 표시/숨김합니다. |

---

## Control Bar 버튼들 (이미지 위)

> 이미지 상단에 위치한 제어 바에서 이미지를 조작할 수 있습니다.

### 줌 제어

| 버튼 | 설명 |
|------|------|
| **축소** | 이미지를 10% 축소합니다. |
| **줌 레벨 표시** | 현재 줌 레벨을 백분율로 표시합니다. (예: 100%, 150%) |
| **확대** | 이미지를 10% 확대합니다. |

### 회전 제어

| 버튼 | 설명 |
|------|------|
| **왼쪽으로 회전** | 이미지를 시계 반대 방향으로 90도 회전합니다. |
| **오른쪽으로 회전** | 이미지를 시계 방향으로 90도 회전합니다. |

### 반전 제어

| 버튼 | 설명 |
|------|------|
| **좌우 반전** | 이미지를 좌우 반전합니다. (미러 이미지) |
| **상하 반전** | 이미지를 상하 반전합니다. (위아래 뒤집기) |

### 보기 제어

| 버튼 | 설명 |
|------|------|
| **원본 크기(1:1)** | 이미지를 원본 해상도로 표시합니다. (100% 줌) |
| **초기화** | 이미지를 초기 상태로 리셋합니다. (모든 변환 해제) |

### 전체 화면

| 버튼 | 설명 |
|------|------|
| **최대화** | 이미지를 전체 화면으로 확대합니다. (F4 키로도 가능) |

### 도움말

| 버튼 | 설명 |
|------|------|
| **도움말** | 사용 팁을 표시하는 모달을 엽니다. |

## Hotkey (키보드 단축키)

| 단축키 | 설명 |
|--------|------|
| **F4** | 최대화 모드 토글 (전체 화면 진입/해제) |
| **Escape** | 도움말 모달 닫기 |

## 팁

- **마우스 조작**:
  - **드래그**: 이미지를 이동할 수 있습니다.
  - **마우스 휠**: 줌 인/아웃이 가능합니다.

- **줌 비율**: 축소/확대 버튼으로 10%씩 조절하거나, 마우스 휠로 미세 조절할 수 있습니다.

- **회전**: 90도 단위로만 회전 가능합니다. 예를 들어, 4번 회전하면 원래 상태로 돌아옵니다.

- **반전**: 좌우 반전과 상하 반전은 독립적으로 작동합니다. 두 가지를 모두 적용할 수도 있습니다.

- **원본 크기 보기**: 고해상도 이미지의 경우, 1:1 보기로 세부 사항을 확인할 수 있습니다.

- **초기화**: 모든 줌, 회전, 반전 설정이 초기 상태로 복귀됩니다.

- **최대화 모드**: 전체 화면으로 이미지를 크게 보려면 최대화 버튼을 사용합니다. F4 키로도 토글할 수 있습니다.

- **정보 보기**: 문서 정보 토글 버튼으로 이미지의 메타데이터를 확인할 수 있습니다.

## 이미지 정보

문서 정보 토글로 표시되는 정보들:

| 정보 | 설명 |
|------|------|
| **파일 크기** | 이미지 파일의 용량 (KB, MB 등) |
| **이미지 크기** | 픽셀 단위의 가로 × 세로 크기 (해상도) |
| **포맷** | 이미지 파일 형식 (PNG, JPG, GIF, BMP, WEBP, SVG 등) |
| **색상 모델** | 색상 표현 방식 (RGB, RGBA, Grayscale, CMYK 등) |
| **비트 깊이** | 픽셀당 비트 수 (8비트, 24비트, 32비트 등) |
| **투명도** | 알파 채널 지원 여부 (Yes/No) |
