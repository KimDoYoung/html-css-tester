# 마크다운 뷰어 (markdown.html)

## 개요

마크다운 문서 뷰어입니다. Markdown을 HTML로 렌더링하며 깃허브 스타일 CSS를 적용합니다. TOC(목차), 검색, 배경색 선택, 다크모드, Raw Markdown 뷰, 다양한 내보내기 형식(HTML, PDF) 등의 기능을 제공합니다.

**주요 특징:**

- Flexmark를 통한 마크다운 렌더링
- 자동 목차(TOC) 생성
- 목차 좌측/우측 위치 선택
- 실시간 검색 기능
- 원본 마크다운 텍스트 뷰 전환
- HTML/PDF 내보내기
- 사용자 선호도 저장

---

## Header 버튼들 (상단 우측)

> 데스크톱 화면에서 파일명 우측에 표시됩니다.

| 버튼 | 설명 |
|------|------|
| **HTML로 다운로드** | 렌더링된 마크다운을 완전한 HTML 파일로 다운로드합니다. |
| **원본 다운로드** | 원본 마크다운 파일을 다운로드합니다. |
| **문서 정보 토글** | 파일 크기, 라인 수, 문자 수, 인코딩 정보를 표시/숨김합니다. |

## Control Bar 버튼들 (문서 위)

> 문서 상단에 위치한 제어 바에서 다양한 설정을 조절할 수 있습니다.

### 검색 기능

| 버튼 | 설명 |
|------|------|
| **검색** | 마크다운 콘텐츠 내에서 실시간 검색을 수행합니다. |

### 폰트 및 외관 설정

| 버튼 | 설명 |
|------|------|
| **폰트 작게 / 기본 / 크게** | 폰트 크기를 조정합니다. (기본값: 16px) |
| **폰트 선택** | 드롭다운으로 폰트를 선택합니다.<br/>- Noto Sans KR (기본)<br/>- 맑은 고딕<br/>- 서울 남산체<br/>- Courier New |
| **배경색 선택** | 배경색을 선택합니다.<br/>- 흰색 (기본)<br/>- 라이트 그레이<br/>- 라이트 옐로우<br/>- 라이트 블루<br/>- 라이트 그린 |
| **줄간 작게 / 기본 / 크게** | 라인 높이를 조정합니다. (기본값: 1.6) |

### 목차(TOC) 설정

| 버튼 | 설명 |
|------|------|
| **목차 위치 선택** | TOC(Table of Contents)를 왼쪽 또는 오른쪽에 표시합니다.<br/>- 왼쪽: 문서 좌측에 목차 고정<br/>- 오른쪽: 문서 우측에 목차 고정 (기본) |
| **목차 토글** | 목차 패널을 표시/숨깁니다. (F2 키로도 가능) |

### 콘텐츠 뷰 전환

| 버튼 | 설명 |
|------|------|
| **Raw Markdown 보기** | 렌더링된 HTML 대신 원본 마크다운 텍스트로 전환합니다. |

### 추가 기능

| 버튼 | 설명 |
|------|------|
| **내보내기** | 드롭다운 메뉴로 내보내기 형식을 선택합니다.<br/>- HTML로 다운로드<br/>- PDF로 다운로드 |
| **전체화면** | 전체 화면 모드로 전환합니다. (⛶) |
| **다크모드** | 어두운 배경 테마를 토글합니다. (🌙) |
| **설정 저장** | 현재 설정(폰트, 크기, 배경색, 줄간, 목차 위치, 다크모드 등)을 localStorage에 저장합니다. |
| **설정 초기화** | 저장된 설정을 초기값으로 리셋합니다. |
| **도움말** | 사용 팁을 표시하는 모달을 엽니다. |

## Hotkey (키보드 단축키)

| 단축키 | 설명 |
|--------|------|
| **Ctrl+S** (Windows/Linux) | 원본 파일 다운로드 |
| **Cmd+S** (macOS) | 원본 파일 다운로드 |
| **Home** | 페이지 맨 위로 스크롤 |
| **F2** | 목차(TOC) 토글 |
| **F4** | 전체화면 모드 토글 |
| **Escape** | 도움말 모달 또는 목차 드로어 닫기 |
| **Enter** (검색창) | 검색 입력창에서 다음 검색 결과로 이동 |

## 팁

- **자동 목차 생성**: 마크다운의 헤딩(# ~ ######)을 기반으로 자동으로 목차가 생성됩니다.
- **목차 클릭 네비게이션**: 목차의 항목을 클릭하면 해당 헤딩으로 자동 스크롤됩니다.
- **설정 자동 로드**: 페이지 로드 시 저장된 설정이 자동으로 적용됩니다.
- **Raw Markdown 뷰**: 마크다운 원본을 확인하고 싶을 때 사용합니다. 문법 강조와 함께 표시됩니다.
- **내보내기**: HTML 또는 PDF로 내보낼 수 있으므로, 다른 포맷으로 문서를 공유할 수 있습니다.
- **다크모드와 배경색**: 다크모드와 배경색 선택은 별도로 작동합니다. 다크모드를 활성화하면 배경색은 무시됩니다.
