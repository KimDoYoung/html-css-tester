CSV 뷰어 가이드

CSV 뷰어는 대용량 데이터를 웹에서 빠르고 편리하게 조회, 검색, 필터링할 수 있는 도구입니다.
상단 컨트롤 바의 다양한 기능을 활용하여 데이터를 효율적으로 분석하세요.

컨트롤 바 기능 상세

1. 보기 설정

Aa

폰트 변경

데이터의 가독성을 위해 폰트(Noto Sans, 맑은 고딕 등)를 변경할 수 있습니다.

14px

글자 크기 조절

+, - 버튼으로 글자 크기를 조절하거나, 가운데 버튼을 눌러 기본값(14px)으로 초기화합니다.

페이지 당 행 수

한 페이지에 표시할 데이터의 개수(10, 25, 50, 100줄)를 선택합니다. '전체'를 선택하면 모든 데이터를 한 번에 표시합니다.

2. 데이터 검색

실시간 검색

검색어를 입력하면 테이블 내의 모든 컬럼에서 일치하는 데이터를 실시간으로 필터링하여 보여줍니다.

  • 검색어 입력 후 엔터 또는 검색 아이콘 클릭
  • X 아이콘을 클릭하여 검색 초기화
  • 검색 결과 건수가 입력창 옆에 표시됩니다.

3. 데이터 관리 도구

컬럼 선택

보고 싶은 컬럼만 체크하여 테이블을 간소화할 수 있습니다. '모두 선택/해제' 기능을 지원합니다.

너비 자동 맞춤

데이터의 길이에 맞춰 모든 컬럼의 너비를 자동으로 최적화합니다.

엑셀 내보내기

현재 보고 있는 데이터(필터링된 결과 포함)를 엑셀 파일(.xlsx)로 다운로드합니다.

4. 화면 및 설정

⛶

전체화면

브라우저 전체 화면으로 뷰어를 확장하여 더 많은 데이터를 한눈에 볼 수 있습니다.

🌙

다크 모드

눈의 피로를 줄이기 위해 어두운 테마로 전환합니다.

설정 저장

현재 변경한 폰트, 컬럼 너비, 숨김 상태 등의 설정을 브라우저에 저장합니다.

설정 초기화

저장된 설정을 삭제하고 뷰어를 초기 상태로 되돌립니다.

정렬 (Sorting)

테이블의 헤더(제목)를 클릭하면 해당 컬럼을 기준으로 오름차순/내림차순 정렬이 가능합니다.

문서 정보

좌측 상단의 화살표 버튼을 클릭하면 파일 크기, 총 행/열 수, 구분자 등의 메타데이터를 확인할 수 있습니다.

원본 다운로드

우측 상단의 '원본 다운로드' 버튼을 통해 CSV 파일을 로컬 PC에 저장할 수 있습니다.
